Detailed Description

Handles (de)serializing of specific fields. Valid types include: bool, byte, char, (u)short, (u)int, (u)long, float, double, ISerializing, and any composite arrays of the former. If the MonoBehaviour being serialized is ISerializing, the ISerializing methods are treated as additions to all fields marked with BitSerialize
