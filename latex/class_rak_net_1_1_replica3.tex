 Without this function, a careless implementation would serialize an object anytime it changed to all systems. This would give you feedback loops as the sender gets the same message back from the recipient it just sent to.\par
 If more than one system can serialize the same object then you will need to override to return true, and control the serialization result from \hyperlink{class_rak_net_1_1_replica3_a6e59b6a75b22441365145bf552f89e09}{Replica3\-::\-Serialize()}.
