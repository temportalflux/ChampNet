\subsection{Detailed Description}
Unique shared ids for each object instance. 

A class you can derive from to make it easier to represent every networked object with an integer. This way you can refer to objects over the network. One system should return true for Is\-Network\-I\-D\-Authority() and the rest should return false. When an object needs to be created, have the the one system create the object. Then have that system send a message to all other systems, and include the value returned from \hyperlink{class_rak_net_1_1_network_i_d_object_a4e2454899578ba16379491fc0aa45bb6}{Get\-Network\-I\-D()} in that packet. All other systems should then create the same class of object, and call \hyperlink{class_rak_net_1_1_network_i_d_object_a5ebfffe7c5bb1a87888b9c3bd940cf35}{Set\-Network\-I\-D()} on that class with the Network\-I\-D in the packet. \begin{DoxySeeAlso}{See Also}
the manual for more information on this. 
\end{DoxySeeAlso}
