Note\-: the file contents are not written, and only the hash if written if {\itshape always\-Write\-Hash} is true always\-Write\-Hash and never\-Write\-Hash are optimizations to avoid reading the file contents to generate the hash if not necessary because the file is missing or has different lengths anyway.
