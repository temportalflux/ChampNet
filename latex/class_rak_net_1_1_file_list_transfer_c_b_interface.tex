Got part of a big file internally in \hyperlink{namespace_rak_net}{Rak\-Net}. 

This is called in one of two circumstances\-: Either the transport layer is returning I\-D\-\_\-\-P\-R\-O\-G\-R\-E\-S\-S\-\_\-\-N\-O\-T\-I\-F\-I\-C\-A\-T\-I\-O\-N, or you got a block via \hyperlink{class_rak_net_1_1_incremental_read_interface}{Incremental\-Read\-Interface} If the transport layer is returning I\-D\-\_\-\-P\-R\-O\-G\-R\-E\-S\-S\-\_\-\-N\-O\-T\-I\-F\-I\-C\-A\-T\-I\-O\-N (see \hyperlink{class_rak_net_1_1_rak_peer_a35e89d9813d6f6e7f2144a6a0437a33b}{Rak\-Peer\-::\-Set\-Split\-Message\-Progress\-Interval()}) then \hyperlink{struct_rak_net_1_1_file_list_transfer_c_b_interface_1_1_file_progress_struct_ab113ec6be9d48acd86f889d316cca04f}{File\-Progress\-Struct\-::iri\-Data\-Chunk} will be 0. If this is a block via \hyperlink{class_rak_net_1_1_incremental_read_interface}{Incremental\-Read\-Interface}, then iri\-Data\-Chunk will point to the block just downloaded. If not using \hyperlink{class_rak_net_1_1_incremental_read_interface}{Incremental\-Read\-Interface}, then you only care about part\-Count and part\-Total to tell how far the download has progressed. Y\-Ou can use first\-Data\-Chunk to read the first part of the file if desired. The file is usable when you get the On\-File callback. If using \hyperlink{class_rak_net_1_1_incremental_read_interface}{Incremental\-Read\-Interface} and you let \hyperlink{namespace_rak_net}{Rak\-Net} buffer the files in memory (default), then it is the same as above. The file is usable when you get the On\-File callback. If using \hyperlink{class_rak_net_1_1_incremental_read_interface}{Incremental\-Read\-Interface} and you do not let \hyperlink{namespace_rak_net}{Rak\-Net} buffer the files in memory, then set allocate\-Ir\-I\-Data\-Chunk\-Automatically to false. Write the file to disk whenever you get On\-File\-Progress and iri\-Data\-Chunk is not 0, and ignore On\-File. \hypertarget{class_rak_net_1_1_file_list_transfer_c_b_interface_a9a7baa02f61c5ef154f40ac6b6d77f84}{\index{Rak\-Net\-::\-File\-List\-Transfer\-C\-B\-Interface@{Rak\-Net\-::\-File\-List\-Transfer\-C\-B\-Interface}!Update@{Update}}
