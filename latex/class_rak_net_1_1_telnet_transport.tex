\subsection{Detailed Description}
Use \hyperlink{class_rak_net_1_1_telnet_transport}{Telnet\-Transport} to easily allow windows telnet to connect to your \hyperlink{class_rak_net_1_1_console_server}{Console\-Server}. 

To run Windows telnet, go to your start menu, click run, and in the edit box type \char`\"{}telnet $<$\-I\-P$>$\char`\"{} where $<$\-I\-P$>$ is the ip address.\par
 of your \hyperlink{class_rak_net_1_1_console_server}{Console\-Server} (most likely the same I\-P as your game).\par
 This implementation always echos commands.
