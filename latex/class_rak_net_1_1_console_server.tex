\subsection{Detailed Description}
The main entry point for the server portion of your remote console application support. 

\hyperlink{class_rak_net_1_1_console_server}{Console\-Server} takes one \hyperlink{class_rak_net_1_1_transport_interface}{Transport\-Interface} and one or more \hyperlink{class_rak_net_1_1_command_parser_interface}{Command\-Parser\-Interface} (s) The \hyperlink{class_rak_net_1_1_transport_interface}{Transport\-Interface} will be used to send data between the server and the client. The connecting client must support the protocol used by your derivation of \hyperlink{class_rak_net_1_1_transport_interface}{Transport\-Interface} . \hyperlink{class_rak_net_1_1_telnet_transport}{Telnet\-Transport} and Rak\-Net\-Transport are two such derivations . When a command is sent by a remote console, it will be processed by your implementations of \hyperlink{class_rak_net_1_1_command_parser_interface}{Command\-Parser\-Interface}
