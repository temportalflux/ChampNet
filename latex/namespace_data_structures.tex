\subsection{Detailed Description}
The namespace \hyperlink{namespace_data_structures}{Data\-Structures} was only added to avoid compiler errors for commonly named data structures As these data structures are stand-\/alone, you can use them outside of \hyperlink{namespace_rak_net}{Rak\-Net} for your own projects if you wish.
