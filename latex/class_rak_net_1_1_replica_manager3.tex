\subsection{Detailed Description}
System to help automate game object construction, destruction, and serialization. 

\hyperlink{class_rak_net_1_1_replica_manager3}{Replica\-Manager3} tracks your game objects and automates the networking for replicating them across the network\par
 As objects are created, destroyed, or serialized differently, those changes are pushed out to other systems.\par
 To use\-:\par
 
\begin{DoxyEnumerate}
\item Derive from \hyperlink{class_rak_net_1_1_connection___r_m3}{Connection\-\_\-\-R\-M3} and implement \hyperlink{class_rak_net_1_1_connection___r_m3_ab05d85100f726378f2ed73c6792a5b46}{Connection\-\_\-\-R\-M3\-::\-Alloc\-Replica()}. This is a factory function where given a user-\/supplied identifier for a class (such as name) return an instance of that class. Should be able to return any networked object in your game. 
\item Derive from \hyperlink{class_rak_net_1_1_replica_manager3}{Replica\-Manager3} and implement \hyperlink{class_rak_net_1_1_replica_manager3_ade7df800d3dfd31bdd44de76172eb912}{Alloc\-Connection()} and \hyperlink{class_rak_net_1_1_replica_manager3_a78b449c8e3caa9a343a7d273f768687c}{Dealloc\-Connection()} to return the class you created in step 1. 
\item Derive your networked game objects from \hyperlink{class_rak_net_1_1_replica3}{Replica3}. All pure virtuals have to be implemented, however defaults are provided for \hyperlink{class_rak_net_1_1_replica3_ad826e04ce6a0db4e32c3d990fd27e943}{Replica3\-::\-Query\-Construction()}, \hyperlink{class_rak_net_1_1_replica3_a79bd207fb79055d2b39943eb1c6b1f58}{Replica3\-::\-Query\-Remote\-Construction()}, and \hyperlink{class_rak_net_1_1_replica3_abc1b2daa5c1a908ddd2d6df44da7e9aa}{Replica3\-::\-Query\-Serialization()} depending on your network architecture. 
\item When a new game object is created on the local system, pass it to \hyperlink{class_rak_net_1_1_replica_manager3_a9046d827471c0be651b54b7efb7b6f28}{Replica\-Manager3\-::\-Reference()}. 
\item When a game object is destroyed on the local system, and you want other systems to know about it, call \hyperlink{class_rak_net_1_1_replica3_a7c6a22d575326cddec3a70eb1c2568f2}{Replica3\-::\-Broadcast\-Destruction()} 
\end{DoxyEnumerate}\par
 At this point, all new connections will automatically download, get construction messages, get destruction messages, and update serialization automatically.
